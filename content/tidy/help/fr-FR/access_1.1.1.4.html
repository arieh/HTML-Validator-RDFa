Accessibilité : [1.1.1.4] : texte alternatif alt douteux (texte de substitution)

Priorité 1

Cause :

Toutes les images exigent un texte équivalent, mais l'attribut alt doit également répondre à certains critères autrement il sera considéré douteux. Par exemple, le texte de l'attribut alt peut être considéré comme inacceptable si un texte de substitution (qui indique une substitution à faire dans le code HTML) est employé au lieu d'un texte équivalent réel (ex : alt="une image ici").

Exemple :

<img src="gifimage.gif" alt="{short description of image}"/>
<img src="gifimage.gif" alt="intérêt de l'image dans la page"/>

Solution :

Indiquez un texte alternatif (attribut 'alt') qui donne une indication sur le sens de la présence de l'image dans la page.
