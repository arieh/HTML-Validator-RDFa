Aide de tidy - Tidy [52] : Le signe > est manquant sur une balise

Cause :

Une balise est obligatoirement composée des signes < et >. Une balise n'est pas écrite correctement.
SI vous utilisez le signe < dans le texte de votre page, veillez à l'écrire sous la forme "&lt;" afin qu'il ne soit pas interprété par l'analyseur.

Exemple :

<p Exemple de texte </p>
Valable dans le cas 0<p, pour une certaine raison.
<p> Exemple de texte </p>
Valable dans le cas 0&lt;p, pour une certaine raison.

Solution :

Insérez le signe > manquant, ou bien codez-le de manière appropriée pour qu'il apparaisse dans votre texte.
