Accessibilité : [10.4.1.3] : Un élément de formulaire a une valeur par défaut invalide (espaces)

Priorité 3

Cause :

Le contenu par défaut dans votre élément de formulaire est composé uniquement d'espaces.

Exemple : 

<form action="">
  <label for="control1">Some text:</label>
  <input id="control1" type="text" value="   "/>
</form>              
<form action="">
  <label for="control1">Some text:</label>
  <input id="control1" type="text" value="valeur par défaut"/>
</form>

Solution :

Insérez un texte par défaut (qui ne soit pas coposé seulement d'espaces) dans votre élément de formulaire.

  • W3C WAI : http://www.w3.org/WAI/
  • Techniques HTML pour le WAI 1.0: http://www.w3.org/TR/WCAG10-HTML-TECHS/
