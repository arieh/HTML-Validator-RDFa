Accessibilité : [5.5.2.1] : élément <caption> manquant dans le tableau

Priorité 2

Cause :

Les résumés sont utilisés pour décrire la nature de la table en une à trois phrases. Un résumé n'est pas toujours nécessaire.

Exemple : 


<table summary="Cette table de données n'a pas de résumé.">

        <tr><td>1</td><td>2</td></tr>
        <tr><td>3</td><td>4</td></tr>

</table>

Solution :

Si besoin, ajoutez un élément <caption> dans votre tableau. S'il n'y a pas de résumé, utilisez l'attribut "title" pour décrire brièvement le rôle du tableau.
