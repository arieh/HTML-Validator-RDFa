Accessibilité : [1.1.8.1] : 'longdesc' nécessaire pour <frame>.

Priorité 1

Cause :

Si une frame ne peut pas être correctement décrite dans sont titre, il faut inclure l'attribut "longdesc" dans la frame pour indiquer un lien vers une information supplémentaire. L'attribut "longdesc" peut aussi être utile pour expliquer la relation entre les différentes frames.

Exemple : 

<frameset rows="100,200">
   <frame src="TableData.htm">
   <frame src="TableLayout.htm" title="this frame title" longdesc="tl.htm">
   <frame src="TableLayout2.htm" title="this frame title" longdesc="tl2.htm">
</frameset>

Solution :

Utilisez l'attribut "longdesc" pour indiquer un lien vers une information supplémentaire.
