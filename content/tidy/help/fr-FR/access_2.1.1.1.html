Accessibilité : [2.1.1.1] : Assurez-vous que l'information n'est pas portée par la couleur seulement

Priorité 1

Cause :

Lorsque la couleur porte de l'information, il faut aussi la rendre disponible par un autre moyen, soit par le contexte, soit par les balises.

Exemple :

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0 Transitional//EN" "http://www.w3.org/TR/REC-html40/loose.dtd">
<html lang="en">
<head>
<title>aert1.0/2.1.1</title>
</head>
<body>
<img src="rex.jpg" alt="big cat">
</body>
</html>

Solution :

Vérifiez que l'information n'est pas portée seulement par la couleur. Assurez-vous aussi qu'il y ait assez de contraste entre les images et la couleur d'arrière-plan.

  • Techniques HTML pour le WAI 1.0: http://www.w3.org/TR/WCAG10-HTML-TECHS/
