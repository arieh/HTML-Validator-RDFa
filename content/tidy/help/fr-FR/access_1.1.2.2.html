Accessibilité : [1.1.2.2] : lien descriptif manquant sur un élément img

Priorité 1

Cause :

Si le texte alternatif (alt) ne peut pas décrire la fonction d'une image ou le but de celle-ci avec une courte description, vous devez proposer une description plus complète. Comme de nombreux navigateurs actuels ne supportent pas l'attribut longdesc, l'utilisation d'un lien descriptif (d-link) assure que les descriptions plus longues sont accessibles à tous les utilisateurs.

Exemple :

<img src="graphique-camembert.jpg" longdesc="graphique-camembert.html" alt="Graphique camembert des dépenses fédérales"/>
<img src="graphique-camembert.jpg" longdesc="graphique-camembert.html" alt="Graphique camembert des dépenses fédérales" /> [<A HREF="longdescsfj.html" TARGET="_top">D</A>] <a href="graphique-camembert.html">D</a>

Solution :

Insérez le lien descriptif.
