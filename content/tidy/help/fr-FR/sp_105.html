Sp 105 : un nom d'attribut doit commencer par une lettre

Cause :

Un attribut utilisé est mal orthographié, son nom commence par un autre caractère qu'une lettre.
Cette erreur apparaît souvent lorsque vous n'avez pas placé la valeur d'un attribut comportant des espaces entre guillemets.

Exemple :

<a href="link.html"">link</a>
<a href="link.html">link</a>

Solution :

Corrigez le nom de l'attribut ou bien placez toutes les valeurs d'attributs entre guillemets.
