Aide de tidy - Tidy [66] : l'élément ... est déjà défini

Cause :

Plusieurs élément ont été déclarés avec le même nom (valeur de l'attribut 'name' ou de l'attribut 'id').

Exemple :

<form name="my_form" action="test1.jsp"></form> <form name="my_form" action="test2.jsp"></form>
<form name="my_form" action="test1.jsp"></form> <form name="my_form2" action="test2.jsp"></form>

Solution :

Modifiez l'un des noms.

Références :

  • W3C WAI : http://www.w3.org/WAI/
  • Techniques HTML pour le WAI 1.0: http://www.w3.org/TR/WCAG10-HTML-TECHS/
