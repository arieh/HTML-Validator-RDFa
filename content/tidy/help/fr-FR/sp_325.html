Sp 325 : Référence à l'entité ... pour laquelle aucun identifiant système ne peut être généré

Cause :

C'est en général une erreur causée par une référence à une entité inconnue ou l'utilisation d'une éperluette simple dans une URL ou du texte. En effet, il faut écrire "&amp;" dans le code HTML pour que & apparaisse à l'écran et ne soit pas interprété comme une entité HTML.

Exemple :

<a href="index.php?a=1&b=2" title="Articles & Didacticiels">
<a href="index.php?a=1&amp;b=2" title="Articles &amp; Didacticiels">

Solution :

Remplacez les caractères spéciaux par les entités correspondantes.
