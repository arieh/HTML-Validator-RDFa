Aide de tidy - Tidy [68] : l'attribut ... est spécifié plusieurs fois

Cause :

L'attribut ... est spécifié plusieurs fois.

Exemple :

<a href="http://exemple.fr" href="http://ex.fr">
<a href="http://exemple.fr">

Solution :

Supprimez l'une des spécifications de l'attribut pointé.
