Accessibilité : [1.1.1.12] : La balise <img> n'a pas de alt (ligne horizontale)

Priorité 1

Cause :

Toutes les images exigent un texte équivalent y compris ceux employés pour indiquer des coupures dans une page Web. Si une image est employée comme ligne horizontale, elle devrait utiliser le texte alternatif alt suivant : alt = "*" ou alt="ligne horizontale".

Exemple :

<img src="bar.gif" width="200" height="5"/>

Solution :

Insérez un texte alternatif.
