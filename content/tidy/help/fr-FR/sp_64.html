Sp 64 : Le type de document déclaré n'autorise pas l'élément ... ici

Cause :

L'élément indiqué se trouve dans un contexte où il n'est pas auorisé par le type de document Cela signifie que vous avez placé l'élément à un endroit où il ne devrait pas être.

Exemple :

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<head>...
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" lang="en" xml:lang="en">
<head>...

Solution :

Déplacer ou supprimer l'élément en cause.
