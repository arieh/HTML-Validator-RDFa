Accessibilité : [10.2.1.2] : Revoir le positionnement de l'étiquette (<label> après <input>)

Priorité 2

Cause :

Vous utilisez un formulaire.

Exemple :

<form action="">
<input id="control1" type="text" />
Text between control and label.
<label for="control1">Some text:</label>
</form>

Solution :

Explicitez visuellement l'association entre un élément de formulaire et son étiquette en s'assurant de leur positionnement relatif dans le code HTML. Ne séparez pas l'étiquette de son élément de formulaire associé par du texte, des images, etc. S'il existe plusieurs éléments de formulaire sur une ligne, les étiquettes doivent être placées juste devant l'élément correspondant, ou bien faites plusieurs lignes.

  • Techniques HTML pour le WAI 1.0: http://www.w3.org/TR/WCAG10-HTML-TECHS/
