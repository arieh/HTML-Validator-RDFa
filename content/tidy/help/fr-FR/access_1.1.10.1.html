Accessibilité : [1.1.10.1] : Un script doit généralement être accompagné d'une section <noscript>

Priorité 1

Cause :

Vous utilisez un script. Il faut que les pages soient utilisables quand les scripts ou tout autre objet programmé sont désactivés ou non supportés.

Exemple :

<script><!-- faire quelque chose --></script>
<script><!-- faire quelque chose --></script>
<noscript><!-- faire quelque chose --></noscript>

Solution :

S'il n'est pas possible de construire une page utilisable sans script, il faut placer une équivalence avec l'élément <noscript>, ou utiliser un script côté serveur au lieu d'un script exécutable sur le poste client.
