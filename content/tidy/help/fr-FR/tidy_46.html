Aide de tidy - Tidy [46] : Eléments ... trop nombreux

Cause :

Un élément a été défini plusieurs fois.

Exemple :

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" lang="en" xml:lang="en">
<head>
<title>Titre</title>
<title>Titre</title>
</head>
...
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" lang="en" xml:lang="en">
<head>
<title>Titre</title>
</head>
...

Solution :

Supprimez les éléments qui ont été déclarés plusieurs fois.
