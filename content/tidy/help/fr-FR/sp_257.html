Sp 257 : Une valorisation d'attribut doit consister en une affectation

Cause :

Une valeur d'attribut n'est pas entre guillemets. Avec les versions antécédentes de HTML, le recours aux guillemets pour encadrer les valeurs d'attributs était conseillé, mais pas obligatoire. Toujours selon les règles de XML, l'utilisation des guillemets n'est plus une proposition, mais bien une obligation

Exemple :

<p class=foo>
<img height=92 …>
<p class="foo">
<img height="92" …>

Solution :

Placer toutes les valeurs d'attributs entre guillemets.

Références :

  • Spécification XHTML 1.0 : Section 4.4: Les valeurs d'attribut doivent toujours être entre guillemets (en anglais)
