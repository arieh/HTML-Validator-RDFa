Accessibilité : [5.3.1.1] : Assurez-vous que la structure des tableaux est correctement linéarisée.

Priorité 2

Cause :

Si une table est utilisée à des fins de mise en page, assurez-vous que le contenu reste intelligible lorsque le tableau est linéarisé (c'est-à-dire lorsque le contenu est converti en simples paragraphes).

Exemple :

<table summary="Ce tableau doit avoir un sens une fois linéarisé">
<caption>Ce tableau a-t-il un sens une fois linéarisé?</caption>
<tr>
<td>Du texte dans une colonne.</td><td>Du texte dans une autre colonne.</td>
</tr>
</table>

Solution :

Il ne faut pas utiliser de tableaux pour la mise en page, à moins que le tableau garde un sens une fois linéarisé. Sinon, si le tableau n'est pas compréhensible, il faut fournir un équivalent alternatif (qui peut être une version linéarisée).
