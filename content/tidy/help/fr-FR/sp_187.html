Sp 187 : Pas de déclaration de type de document, parse sans validation

Cause :

Il n'y a pas de déclaration de type de document au début de votre fichier HTML. Le parseur ne peut donc pas travailler correctement.

Exemple :

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

Solution :

Spécifiez une déclaration de type de document pour que le parseur puisse valider votre fichier HTML.

Références :

  • W3 validator: http://validator.w3.org/
