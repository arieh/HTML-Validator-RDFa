Choisissez votre algorithme

Le W3C a écrit 2 algorithmes qui vérifient la conformité d'une page aux normes HTML/XHTML.
Les deux algorithmes sont embarqués par HTML Validator dans Firefox.


HTML Tidy
Parseur SGML (OpenSP)
Description C'est un algorithme destiné à vérifier la syntaxe HTML. C'est le même algorithm qui est utilisé par le valideur en ligne du W3C (validator.w3c.org)
Meilleur pour HTML XHTML
Règles d'accessibilité WAI niveau 1,2,3 -
Nettoyage des pages Oui -
