Sp 378 : Pas de type de document

Cause :

Vous utilisez une déclaration de type de document, mais il ne contient pas d'identifiant système (qui désigne le DTD utilisé). L'indication de l'identifiant système est obligatoire dans les langages XML.

Exemple :

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" >
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

Solution :

Ajoutez une URL pointant vers le DTD utilisé.
