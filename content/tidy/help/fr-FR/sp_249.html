Sp 249 : Les DTD particuliers ne sont pas autorisés

Cause :

Les DTD particuliers ne sont autorisés que si le flag CONCUR a la valeur YES ou si le flag EXPLICIT a la valeur YES. Ce message peut apparaître lorsque deux DTD sont déclarés dans un seul document.

Exemple :

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd" > <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/loose.dtd"> <html> ....
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd" > <html> ....

Solution :

Supprimez l'un des DTD déclarés.
