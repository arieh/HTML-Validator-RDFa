Sp 111 : Une valeur d'attribut ne peut être spécifiée qu'après le 'VI delimiter'

Cause :

Vous avez oublié le signe 'égal' entre le nom de l'attribut et sa valeur. 'VI delimiter' est un nom technique qui désigne le signe 'égal'. La syntaxe correcte est :attribut="valeur".

Exemple :

<div class"post">
<div class="post">

Solution :

Ajoutez le signe 'égal' entre la nom de l'attribut et sa valeur.
