Accessibilité : [8.1.1.1] : assurez vous que les éléments programmables sont accessibles (script)

Priorité 1

Cause :

Vous utilisez un script.

Exemple :

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0 Transitional//EN" "http://www.w3.org/TR/REC-html40/loose.dtd">
<html lang="en">
<head>
<title>Accessibilité 8.1.1</title>
</head>
<body>
<script type="a"><!-- this is the script --></script>
</body>
</html>

Solution :

Assurez vous que le contenu de la page est accessible lorsque les scripts sont désactivés ou ne sont pas supportés. Faites des scripts directement accessibles ou compatibles avec les outils d'accessibilité si la fonctionalité est importante. Créez une section <noscript>dans le document et utilisez des gestionnaires d'événements indépendants des périphériques. Si ce n'est pas possible, fournissez le contenu sur une page alternative.
