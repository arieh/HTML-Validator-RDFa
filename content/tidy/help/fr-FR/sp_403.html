Sp 403 : référence à une entité externe dans la valeur de l'attribut

Cause :

Ce message peut apparaître si vous faîtes une référence erronée dans une valeur de l'attribut.

Exemple :

<a href="http://www.domain.com/cgi?x=123&y=456">
<a href="http://www.domain.com/cgi?x=123&amp;y=456">

Solution :

Modifiez les entités dans les valeurs d'attribut.
