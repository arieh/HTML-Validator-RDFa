Aide de tidy - Tidy [37] : Les mentions SYSTEM, PUBLIC, W3C, DTD, EN doivent être en majuscules

Cause :

La déclaration de type de document est obligatoire. Le type de document (DOCTYPE) défini le langage utilisé dans le document. Certains mots particuliers de cette déclaration doivent être écrits en majuscules.

Exemple :

<!doctype html public "-//w3c//dtd xhtml 1.0 Strict//en" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" lang="en" xml:lang="en">
<head>
<!DOCTYPE HTML PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" lang="en" xml:lang="en">
<head>

Solution :

Vérifiez que les mots SYSTEM, PUBLIC, W3C, DTD, EN sont écrits en majuscule dans la déclaration de type de document.
