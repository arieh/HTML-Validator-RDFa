Accessibilité : [5.5.1.2] : l'attribut "summary" dans <table> est invalide (vide).

Priorité 3

Cause :

Les tableaux doivent contenir un attribut "summary". Le résumé doit décrire le lien entre les données du tableau (entre elles) et entre le contenu du tableau et le reste du document.

Exemple : 

<table summary="">
	<caption>Ce tableau n'a pas de "summary"</caption>

	<tr><td>1</td><td>2</td></tr>
	<tr><td>3</td><td>4</td></tr>

</table>            
<table summary="ce tableau montre d'intéressantes données sur ...">
	<caption>Ce tableau n'a pas de "summary"</caption>

	<tr><td>1</td><td>2</td></tr>
	<tr><td>3</td><td>4</td></tr>

</table>

Solution :

Insérez un attribut "summary" non vide dans votre déclaration de tableau.
