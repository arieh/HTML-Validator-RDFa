Accessibilité : [9.3.1.1] : <script> inaccessible au clavier (onMouseDown)

Priorité 2

Cause :

Vous avez utilisé une portion de script qui est activée par la souris uniquement.

Exemple :

<p onMouseDown="Blabla"></p>

Solution :

Incluez un mécanisme redondant (comme une entrée clavier) si des gestionnaires d'évènements dépendants de l'interface sont utilisés pour invoquer un script. Le gestionnaire d'évènement onMouseDown devrait être accompagné de onKeyDown et inclus dans le même élément.
