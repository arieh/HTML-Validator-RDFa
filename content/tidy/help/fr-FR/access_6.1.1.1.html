Accessibilité : [6.1.1.1] : Les feuilles de style nécessitent une vérification (lien)

Priorité 1

Cause :

Les styles doivent être externalisés. Vous utilisez une feuille de style externe, assurez-vous que la feuille de style pointée existe bien.

Exemple :

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<title>Accessibilité 6.1.1 - feuille de style externe</title>
<link rel="stylesheet" type="text/css" href="http://www.w3.org/StyleSheets/TR/W3C-WD">
</head>
<body>
</body>
</html>

Solution :

Assurez-vous que l'information demeure accessible lorsque les feuilles de style sont désactivées ou non prises en charge. Incluez une méthode de présentation des informations non visuelle et organisez les feuilles de style rationnellement.
Assurez-vous que la feuille de style déclarée dans le code HTML existe bien.

Références :

  • W3C WAI : http://www.w3.org/WAI/
  • Techniques HTML pour le WAI 1.0: http://www.w3.org/TR/WCAG10-HTML-TECHS/
