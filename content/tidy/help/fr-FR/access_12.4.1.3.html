Accessibilité : [12.4.1.3] : associer explicitement les étiquettes avec les éléments de formulaire (id)

Priorité 2

Cause :

Vous utilisez un formulaire. Vous n'utilisez pas l'attribut "id" correctement.

Exemple :

<form action="">
<label>Some text:</label>
<input id="control1" value="****" type="text" />
</form>
<label for="control1">Some text:</label>
<input id="control1" value="****" type="text" />

Solution :

Il faut utiliser l'attribut "id" afin d'associer explicitement l'étiquette avec l'élément de formulaire correspondant. Les valeurs des attributs "for" et "id" doivent être égales. Il est possible d'associer plusieurs étiquettes avec un même élément en créant des références multiples à l'aide de l'attribut "for".
