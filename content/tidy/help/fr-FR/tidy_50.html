Aide de tidy - Tidy [50] : L'attribut ... n'a pas de valeur assignée

Cause :

Un attribut est déclaré, mais il n'a pas de valeur affectée.

Exemple :

<a href>Mon Lien</a>
<a href="mon_lien.html">Mon Lien</a>

Solution :

Donnez une valeur à l'attribut.
