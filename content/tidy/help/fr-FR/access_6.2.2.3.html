Accessibilité : [6.2.2.3] : les équivalences textuelles nécessitent une mise à jour (object).

Priorité 1

Cause :

A chaque fois que vous modifiez le contenu dynamique, les équivalents textuels doivent être mis à jour. Vous utilisez un objet (marqué par les balises <object>) qui peut inclure du contenu dynamique, si c'est le cas, maintenez les équivalents textuels à jour dans la section de l'objet lui-même.

Exemple :

<object><!-- object délivrant du contenu dynamique -->
Texte alternatif. </object>

Solution :

Les équivalences pour le contenu dynamique doivent être mises à jour.
