Sp 42 : déclaration de type inconnue

Cause :

Vous tentez de déclarer une entité qui n'est pas prévue.
Cette erreur est susceptible d'apparaître si vous avez une syntaxe erronée pour écrire les commentaires :<!commentaire invalide>. La syntaxe correcte est :<!-- commentaire correct -->.

Exemple :

<!commentaire invalide>
<!p>Paragraphe mal mis en commentaire.</p>
<!-- commentaire correct -->
<!--p>Paragraphe correctement mis en commentaire.</p-->

Solution :

Corrigez la déclaration du commentaire.
