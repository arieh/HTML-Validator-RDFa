Accessibilité : [3.3.1.1] : Il faut utiliser des feuilles de style pour la mise en page et la présentation

Priorité 2

Cause :

Utilisez des feuilles de style (CSS) pour contrôler la mise en page (positionner le contenu) et la présentation

Exemple :

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0 Transitional//EN" "http://www.w3.org/TR/REC-html40/loose.dtd">
<html lang="en">
<head>
<title>aert1.0/3.3.1</title>
</head>
<body>
Does not use stylesheets.
</body>
</html>

Solution :

Créez une feuille de style et faites-y référence dans votre document HTML (balise <meta> dans la section <head>).

Références :

  • W3C WAI : http://www.w3.org/WAI/
  • Techniques HTML pour le WAI 1.0: http://www.w3.org/TR/WCAG10-HTML-TECHS/
