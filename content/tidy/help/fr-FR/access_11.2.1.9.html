Accessibilité : [11.2.1.9] : Les balises obsolètes sont interdites en XHTML strict (strike)

Priorité 2

Cause :

Vous utilisez la balise <strike> qui est obsolète et n'existe plus dans le XHTML.

Exemple :

<strike>
</strike>

Solution :

Remplacez la balise <strike> par les styles CSS correpondants ( "text­decoration : line-through;") et respectant les normes les plus récentes.

Références :

  • W3C WAI : http://www.w3.org/WAI/
  • Techniques HTML pour le WAI 1.0: http://www.w3.org/TR/WCAG10-HTML-TECHS/
