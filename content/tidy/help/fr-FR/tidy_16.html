Aide de tidy - Tidy [16] : la valeur de l'attribut doit être entre guillemets

Cause :

Vous donnez la valeur d'un attribut sans spécifier la valeur entre guillemets.

Exemple :

<span class=maclasse>...
<span class="maclasse">...

Solution :

Spécifiez la valeur de l'attribut entre guillemets.
