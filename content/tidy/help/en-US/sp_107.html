The name and VI delimiter can be omitted from an attribute specification only if SHORTTAG YES is specified

Cause:

You have specified an attribute without giving an equals sign and its value. "VI delimiter" is a technical term for the equals sign. A common cause for this error message is the use of "Attribute Minimization" in document types where it is not allowed, in XHTML for instance.</p>


Example:

<option selected>Option</option>
<option selected="selected">Option</option>

Solution:

Add in the equals sign and an appropriate value for the attribute.
