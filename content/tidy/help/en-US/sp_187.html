No document type declaration; will parse without validation

Cause:

No document type specified at the begin of the document.

Example: 

<html>
   <head>
      	<title>my_title</title>
      	...
   </head>
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
<html>
   <head>
      	<title>my_title</title>
      	...
   </head>

Solution:

Add a doctype before the <html> tag.

References:

  • W3C spec: http://www.w3.org/TR/html4/struct/global.html#h-7.2
