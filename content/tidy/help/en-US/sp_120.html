Normalized length of attribute value literal must not exceed LITLEN (...); length was ...

Cause:

This error almost always means that you've forgotten a closing quote on an attribute value, meaning that everything following that point might be interpreted as part of the attribute's value. SGML defines the acceptable length of string literals within HTML. Note that the position indicator in the error message points to where the attribute value ended — in this case, the "joe.gif" line.

Example: 

<img src="fred.gif>
<!-- 50 lines of stuff -->
<img src="joe.gif">
<img src="fred.gif">
<!-- 50 lines of stuff -->
<img src="joe.gif">

Solution:

Close the quote.
