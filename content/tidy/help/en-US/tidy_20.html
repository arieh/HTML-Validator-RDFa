Replace obsolete element

Cause:

Obsolete elements (like <menu> or <dir>) used.

Example:

<menu><li>File</li></menu>
<ul><li>File</li></ul>

Solution:

Such elements should not be used anymore, for up-to-date browsers will not always support these elements. Use up-to-date elements instead.

References:

  • W3C spec: http://www.w3.org/TR/html4/appendix/changes.html#h-A.3.1.2
  • Tags definitions: http://www.w3.org/TR/html4/struct/lists.html#edef-MENU
